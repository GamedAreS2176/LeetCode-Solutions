# LeetCode Java Solutions

This repository contains Java solutions for a growing set of LeetCode problems. Each file is usually named after the problem number, which makes it easy to find a specific solution quickly.

## Repository Layout

- `Code1.java` through `Code704.java` contain individual LeetCode solutions.
- Most files are self-contained and are intended to be compiled one at a time.
- Some files use the class name `Solution` to match the LeetCode submission format.
